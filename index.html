SAMPLER TYPEWRITER
Upload your own image files using button at lower left (may need to scroll).
Switch between source images using number keys or '=' key to advance (will wrap back to first image).
Select a portion of the source image to sample it. Samples are automatically sequenced on the right.
ENTER to start a new line.
SHIFT plus letter keys a-z, or spacebar, to map the last sample to a key. Typing that key will repeat the sample.
Arrow keys will flip the most recent sample horizontally or vertically.
At the start of a new line, drag the rectangular handle at the bottom of the cursor to change line height.
